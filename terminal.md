## объединение команд
#### `;` Command1 -> Command2
#### `&&` Command1 {true} -> Command2
#### `||` Command1 {false} -> Command2 {true} | Command1 {true} -> Command2 {false}
#### `()` mkdir $(date "+%Y-%m-%d")
#### `|` результат Command1 передан Command2
#### `>` записать данные в файл
#### `>>` записать данные в конец файла
#### `<` прочитать данные из файла
